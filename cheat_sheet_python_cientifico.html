Cheat Sheet Python Científico

Resumen profesional para trabajo numérico, visualización científica, análisis tabular y flujos reproducibles con NumPy, Matplotlib y Pandas.

Bibliotecas base

NumPy Matplotlib Pandas Jupyter
  • NumPy permite trabajar con arrays, cálculo vectorizado, álgebra lineal, integración y discretización.
  • Matplotlib se usa para gráficos 2D, mapas de calor, contornos, series temporales y figuras listas para informes o papers.
  • Pandas facilita el manejo de tablas, CSV, Excel, fechas, filtros y análisis exploratorio.
  • En flujo científico real: NumPy calcula, Matplotlib visualiza y Pandas organiza y resume.

18. Modelamiento matemático con NumPy

Herramienta ¿Qué hace? Sintaxis Ejemplo
Malla 1D Discretiza un dominio continuo. np.linspace(a, b, n) x = np.linspace(0, 1, 100)
Malla 2D Construye dominio bidimensional. np.meshgrid(x, y) X, Y = np.meshgrid(x, y)
Derivada numérica Aproxima derivadas discretas. np.gradient(f, dx) df = np.gradient(f, dx)
Integral numérica Aproxima integrales. np.trapz(f, x) I = np.trapz(f, x)
Campo escalar 2D Evalúa funciones sobre malla. Z = f(X, Y) Z = np.sin(X)*np.cos(Y)
Mapa de calor Visualiza soluciones 2D. plt.imshow(Z) plt.imshow(Z, origin="lower")
Contornos Representa niveles de una función. plt.contour(X,Y,Z) plt.contourf(X,Y,Z,20)
Guardar resultados Exporta datos numéricos. np.save, to_csv np.save("Z.npy", Z)
Flujo típico en modelamiento:
discretizar dominio → evaluar modelo → derivar/integrar → visualizar → exportar resultados
Consejo clave: evita loops en Python cuando puedas. Usa operaciones vectorizadas con NumPy para eficiencia.
