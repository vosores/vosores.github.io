Cheat Sheet Linux (Ubuntu)

Resumen profesional de terminal, archivos, SSH, permisos, análisis de texto, procesos, red y administración básica.

Conceptos clave

Linux libre Multiusuario Multitarea CLI
  • Linux organiza todo desde el directorio raíz /.
  • El directorio personal suele estar en /home/usuario.
  • Muchos procesos administrativos requieren sudo.
  • El acceso remoto seguro normalmente se realiza con ssh.

Directorios importantes

  • /bin ejecutables básicos
  • /boot arranque
  • /dev dispositivos
  • /etc configuración
  • /home usuarios
  • /lib bibliotecas
  • /media medios montados
  • /opt aplicaciones extra
  • /tmp temporales
  • /usr programas compartidos
  • /var datos variables y logs
  • /proc procesos y kernel
Acceso remoto: ssh usuario@servidor   |   Ejemplo: ssh username@ima.ucm.cl
