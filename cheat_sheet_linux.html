Cheat Sheet Linux (Ubuntu)

Resumen de terminal, archivos, SSH, permisos, análisis de texto y procesos.

Conceptos clave

Linux libre Multiusuario Multitarea Multiplataforma
  • El sistema operativo permite interactuar con el hardware y ejecutar aplicaciones.
  • En Linux todo cuelga del directorio raíz /.
  • El directorio personal del usuario suele estar en /home/usuario.
  • El acceso remoto seguro se hace con ssh.

Directorios importantes

  • /bin ejecutables básicos
  • /boot arranque del sistema
  • /dev dispositivos
  • /etc configuración
  • /home usuarios
  • /lib bibliotecas
  • /media dispositivos montados
  • /opt aplicaciones
  • /tmp temporales
  • /usr aplicaciones compartidas
  • /var datos variables
  • /root administrador
  • /proc kernel y procesos
Acceso remoto: ssh remote_username@remote_host   |   Ejemplo: ssh username@ima.ucm.cl
